## 📌 Featured Projects

### 🤖 [Swaylytics — AI-Powered Autonomous Data Analyst](https://www.loom.com/share/2ddf284bccb44e9da42b5b5c5bcd61ad) <sub>*(Loom demo)*</sub>
A multi-agent system that automates end-to-end exploratory data analysis on user-uploaded datasets.
- **LoRA-fine-tuned DeepAnalyze-8B** worker + **Gemini 3 Flash** orchestration via **LangGraph**  
- Outperformed **GPT-4-turbo by ~15%** on the **DS-1000** benchmark with a smaller 8B model  
- 4 specialized agents — *planner, analyzer, visualizer, critic* — with iterative self-correction  
- Stack: PyTorch, FastAPI, Docker, LangGraph, LoRA

### 🧠 Agentic RAG System with Iterative Self-Correction
An intelligent knowledge retrieval framework — work behind my FLAIRS-39 full paper.
- Multi-agent architecture: retrieval, reasoning, planning, and verification agents  
- **92% routing accuracy** on complex multi-hop queries  
- **Pre-Act planning** + automated question decomposition → 35% fewer incomplete responses  
- Stack: LangChain, LangGraph, ChromaDB, OpenAI Embeddings, FastAPI

### 🛡️ Edge IoT Threat Detection with Tiny LLMs
Fine-tuned 3 SLMs for on-device security classification — work behind my FLAIRS-39 short paper.
- **23.37 pp accuracy improvement** over XGBoost baseline (76.93% vs 53.56%)  
- **129ms inference latency** at 14.25 tokens/sec on edge hardware  
- Automated threat mitigation pipeline mapping to **MITRE CAPEC** strategies  
- Stack: PyTorch, Hugging Face Transformers, LoRA/PEFT

### 🌍 Panvel Municipal Corporation — Solid Waste Management Portal
Civic platform serving **5,000+ daily users** across desktop and mobile.
- Real-time data sync via RESTful APIs → 40% reduction in reporting delays  
- 29% page load time improvement through code-splitting, lazy loading, caching  
- Dockerized + CI/CD on GitHub Actions + Jenkins  
- Stack: React.js, Node.js, AWS, Docker
